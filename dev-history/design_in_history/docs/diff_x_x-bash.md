# x与x-bash的区别

x-bash用来简化bash脚本的编写
x也可以简化bash脚本编写

## x

1. 能够使用x-bash
2. 能够跨平台
3. 可通过SDK与S3，获取数据
4. 减少依赖系统中现有的非标准的二进制工具，缺失则由x或者static-build来提供

## x-bash

定位：简单，一目了然的设计

1. 标准库
2. 稳定，避免增加过多新功能
3. 可信赖
4. 放置教程与cookbook，快速学习bash
5. 自动查阅

以src为主，用来增强bash内交互的使用体验

x-bash与x并不重复设计，两者互相集成。

# 冲突：

x中大部分运行功能，x-bash都能通过组合bash命令完成
